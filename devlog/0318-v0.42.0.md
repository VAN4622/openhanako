# v0.42.0 — 兼容技能 + 上下文感知 + Bug Fixes

## 新功能

### 兼容技能（Compatible Skills）
- 设置页新增「兼容技能」section，自动发现 Claude Code / Codex / Pi / OpenClaw 等工具安装的技能
- 支持手动添加自定义外部技能目录
- 书桌面板新增技能快捷区，展示当前 agent 启用的技能并支持开关
- 外部技能目录支持 file watcher 热加载
- 运行期间新出现的外部目录会自动集成（无需重启）

### 上下文压缩通知
- InputArea 新增 ContextRing 环形指示器，实时显示上下文 token 用量
- 支持自动压缩 / 手动压缩，前端同步显示压缩状态
- 切换会话时自动刷新 context ring

### QQ Adapter 稳定性
- 心跳 ACK 超时检测，未收到 ACK 时强制重连
- 长连接保活超过 5 分钟后重置重连计数
- Token 刷新连续失败时通知前端状态
- 消息发送全链路错误日志（C2C → 群 → 频道）

## Bug Fixes

- **巡检间隔不生效**（#69）：修改间隔后重建 heartbeat 实例
- **定时任务模型找不到**（#65）：executeIsolated 模型 fallback 到默认模型
- **Agent 误认 OpenClaw**（#80）：system prompt 加平台标识
- **删除供应商后模型残留**（#72）：删 provider 时同步清理 models.json + favorites
- **删除 provider 时 models.json 残留**：provider 删除逻辑增加 models.json 清理

- learn_skills 默认值补全
